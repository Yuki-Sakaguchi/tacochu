# 🌮 Tacochu 🌮
毎週火曜日はタコスの日！  
タコスを作るためのWebアプリです。

## 開発
```bash
npm run dev
```

```bash
supabase start
supabase stop
```

## 技術スタック
- T3 Stack
  - Next.js
  - TypeScript
  - Prisma
  - tRPC
  - NextAuth.js
  - Tailwind CSS
- Supabase
  - Database（PostgreSQL）
  - Auth
- Docker
  - Supabase のローカル環境の実行
